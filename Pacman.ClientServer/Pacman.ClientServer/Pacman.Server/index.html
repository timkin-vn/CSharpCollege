🎮 Pacman Server API

Backend сервер для игры Pacman

✅ Сервер работает и готов к приёму запросов

📡 API Endpoints

GET /api/users
Получить список всех пользователей
POST /api/users
Создать нового пользователя
GET /api/games/{userId}
Получить все игры пользователя
POST /api/games/create
Создать новую игру
GET /api/games/{id}/state
Получить состояние игры
POST /api/games/{id}/move
Выполнить ход в игре
DELETE /api/games/{id}
Удалить игру
GET /api/leaderboard
Получить таблицу лидеров

Сервер работает на localhost:44375

База данных: PostgreSQL
