Разработка БД PostgreSQL с использованием EF для .NET Framework 4.*

1. Версия Npgsql не выше 6.0.13.
2. Добавить пакет EntityFramework6.Npgsql.
3. Выровнять версии всех устанолвленных пакетов.
4. Созать в pgAdmin новую БД (пустую!).
5. Указать строку подключения.
6. В проекте с репозиториями разработать DbContext со всеми сущностями.
7. Открыть окно Package Manager Console (Tools -> NuGetPackage Manager -> Package Manager Console). В списке Default project выбрать проект с репозиториями.
8. Выполнить команду Enable-Migrations.
9. Выполнить команду Add-Migration <имя миграции>.
10. Выполнить команду Update-Database. Шаги 9 и 10 повторять по мере необходимости.
11. Разработать репозитории, протестировать.
