Разработка БД PostgreSQL для .NET Framework 4.*

1. В административном приложении (pgAdmin) создать БД и разработать структуру таблиц.
2. Подключить к проекту пакет Npgsql (Tools -> NuGetPackage Manager -> Manage NuGet Packages for Solution) версии не выше 8.0.7.
3. Указать строку подключения в app.config.
4. Разработать репозитории.
5. Разработть юнит-тесты, добиться их прохождения.
6. Доработать бизнес-сервисы:
	6.1. Использовать репозитории для извлечения данных
	6.2. Получать вызовы с указанием id вместо полной бизнес-модели (где применимо)
7. Доработать представления.
